# 📊 EngageMetrics

An AI-assisted student engagement tracking system that analyzes participation, quiz performance, and feedback to generate actionable insights for educators.

---

## 🚀 Overview

EngageMetrics is designed to help educators monitor and improve student engagement in online classes. It collects post-class data, calculates engagement scores, analyzes feedback sentiment using AI, and identifies at-risk students using rule-based logic.

The system leverages no-code tools and AI integration to provide structured insights without using complex machine learning models.

---

## 🎯 Features

### 1. 📥 Engagement Data Collection
- Collects data using Google Forms:
  - Participation rating
  - Quiz score percentage
  - Chat activity summary
  - Written feedback

---

### 2. 🧮 Engagement Scoring System
- Rule-based scoring using:
  - Participation (40%)
  - Quiz Performance (40%)
  - Feedback Sentiment (20%)
